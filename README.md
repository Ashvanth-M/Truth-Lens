# 🛡️ TruthLens

> **Agentic Exam Integrity Operating System for India**

TruthLens is an AI-powered examination integrity platform designed to secure the entire examination lifecycle — from identity verification before the exam to post-exam forensic investigations. Built for **FAR AWAY 2026**, TruthLens aims to restore trust in high-stakes examinations while preserving student privacy and fairness.

---

## 🌟 The Problem

India conducts some of the world's largest examinations, including **NEET, JEE, UPSC, CBSE Boards, and State Entrance Exams**. Every year, millions of students compete for limited opportunities.

However, the examination ecosystem faces major challenges:

* 🎭 Proxy impersonation
* 📱 Phone and hidden device usage
* 📝 Organised cheating networks
* 🔓 Paper leaks
* ☁️ Privacy concerns from cloud-based proctoring
* 📶 Dependence on stable internet connectivity
* ⚖️ Bias caused by unfair AI-based auto-disqualification

Existing solutions only monitor students during exams and fail to address the complete integrity lifecycle.

---

## 💡 Our Solution

TruthLens is an **Agentic Exam Integrity Operating System** that secures examinations across all stages:

```
Before Exam
     ↓
Identity Vault
     ↓
During Exam
     ↓
Live Guard
     ↓
After Exam
     ↓
Pattern Investigator
     ↓
Audit Intelligence
```

Unlike traditional proctoring software, TruthLens provides **end-to-end examination integrity**.

# 🚀 Core Features

## 🛡️ Identity Vault

Ensures that the right student takes the exam.

### Features

* Student Registration
* Face Verification
* Voice Verification
* Keystroke Dynamics
* Aadhaar-assisted verification
* Zero-Knowledge Proof generation
* Offline enrollment support

### Why it matters

Instead of storing sensitive biometric data, TruthLens verifies identity while protecting student privacy.

---

## 👁️ Live Guard

AI-powered exam monitoring that runs entirely on-device.

### Features

* Continuous Face Matching
* Gaze Tracking (MediaPipe)
* Phone Detection
* Chit Detection
* Second Person Detection
* Confidence-based anomaly scoring
* Human review queue
* Offline AI inference

### Fairness First

TruthLens never automatically disqualifies students.

Every anomaly follows:

```
AI Detection
    ↓
Human Review
    ↓
Decision Recording
```

---

## 🕸️ Pattern Investigator

Post-exam forensic intelligence engine.

### Features

* Collusion Graph Analysis
* Cross-centre anomaly detection
* Leak probability scoring
* Suspicious answer similarity detection
* Investigation recommendations
* Timeline playback

### Why it matters

TruthLens investigates organised cheating instead of focusing only on individual incidents.

---

## 📄 Audit Intelligence

Generates board-ready integrity reports.

### Features

* Executive Summaries
* Integrity Scores
* Evidence Compilation
* Recommendations
* Claude-powered explanations
* PDF Export

---

## 🤖 Agentic Orchestrator

The intelligence layer connecting every phase.

### Responsibilities

* Escalates investigations
